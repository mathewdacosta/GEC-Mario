# SDL Mario Bros

## About

A recreation of the classic Mario Bros game.

This was created using C++ and SDL for a university assignment.

## Controls

- W: Mario Jump
- A/D: Move Mario left/right

- Up Arrow: Luigi Jump
- Left/Right Arrow: Move Luigi left/right

- Escape: Exit the game
